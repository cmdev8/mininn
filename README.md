# Toy neural network
Not for production use!

```
Usage of ./mininn:
  -dataset string
    	Dataset csv path
  -learning-rate float
    	Learning rate (default 0.01)
  -modelPath string
    	ModelPath (default "model.bin")
  -n-classes int
    	Number of classes
  -n-epochs int
    	Number of epochs (default 1000)
  -n-features int
    	Number of features
  -n-hidden int
    	Number of hidden nodes
  -predict
    	Prediction mode
  -train
```
